> [!definition] Int Data Type  
> In Python, the `int` data type is an integer of any size.  
  
```python  
type(203203423040) # int  
type(7 - 14) # int  
```  
  
Ints have arbirtrary precision in Python, meaning that calculations with them will be precise.
