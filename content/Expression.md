> [!definition] Expression
> An **expression** is a combination of values, operators, and functions that **evaluates** to some **value**.

```Python
23 # Prints 23 when run as a cell in a Jupyter notebook
```

---

```Python
-15 + 2.718 # Evaluates to -12.282
```

---

```Python
4 ** 3 # 64
```
